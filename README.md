# 🎮 Level Up!

Gioco di ruolo a turni in stile dungeon crawler sviluppato in Java con JavaFX.
Scegli un eroe tra tre classi, esplora un dungeon procedurale di 5 piani,
combatti nemici, raccogli armi e affronta boss sempre più potenti.

---

## 🚀 Come eseguire il progetto

### Prerequisiti

- Java 25 (LTS)
- Gradle 8.10.2

### Build

```bash
./gradlew build
```

### Esecuzione

```bash
./gradlew run
```
---

## 🎮 Come si gioca

- **WASD** per muovere l'eroe nella mappa
- **Mouse** per interagire con i pulsanti (attacca, usa pozioni, fuggi, ecc.)

---

## 🤖 Uso di strumenti di AI

Ho utilizzato **Claude (Anthropic)** come supporto durante lo sviluppo, principalmente per:

- Chiarire dubbi su concetti teorici (es. applicazione dei principi SOLID, pattern MVC)
- Suggerimenti su come strutturare alcune parti del codice
- Supporto nel debugging di bug logici
- Autocompletamento di codice ripetitivo (getter, setter, switch su enum)

Tutto il codice è stato compreso, revisionato e testato personalmente.
Le decisioni architetturali, la struttura del progetto e i test sono frutto di ragionamento proprio.
